# 💫 Wish-List App

> - **" 간절히 바라면 무엇이든 이루어진다! "**
> - 자신의 위시리스트를 작성하고, 라우트를 나눠서 상세페이지에서 작성 날짜를 알 수 있게 함.
> - localStorage 에 저장하고, `redux`로 상태관리를 함.
> - 모바일 화면에 최적화됨. (추후 desktop 버전도 다시 스타일링 예정)

# 🌐 [배포 링크](https://thisisyjin.github.io/wish-list/)
- `gh-pages`로 배포. 
```  
$ gh-pages -d build
```

## ⚙️ Setting

```
$ git clone https://github.com/thisisyjin/wish-list.git

$ cd wish-list
$ npm install

$ npm start || yarn start
```

<br>
